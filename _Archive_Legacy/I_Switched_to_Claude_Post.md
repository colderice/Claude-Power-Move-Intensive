THE SWITCH PROMPT

Paste this into ChatGPT, Gemini, or wherever you've been working:

Export everything you know about me into a structured system prompt I can use with any AI model. Preserve my words verbatim where possible, especially for instructions, preferences, and corrections to your behavior.

Categories (output in this order):
1. Core Identity — Who I am, what I do, key context. Name, location, background, and the one-line answer to "what does this person actually do in the world."
2. Instructions — Rules I've explicitly asked you to follow. Tone, format, style, "always do X", "never do Y", and corrections to your behavior. Only pull from stored memories, not the current conversation.
3. Communication Style — How to talk to me. Language level, tone preferences, structure I like, what to avoid. If I've corrected you or pushed back on a style choice, include that.
4. Work & Projects — Current roles, responsibilities, and tools I use. One entry per project. Include what it does, current status, and any key decisions or frameworks attached to it.
5. Thinking Style & Frameworks — How I approach problems. Decision-making patterns, mental models I use, frameworks I've referenced, how I like to structure my thinking.
6. Knowledge Level — Where I'm expert vs. where I need things explained. Domains I own deeply vs. areas I'm still learning.
7. Values & Priorities — What drives my decisions. What I'm focused on right now. What matters to me in work and life.
8. Common Requests — Recurring themes, patterns in our conversations, things I typically ask for help with.
9. Avoid — What NOT to do. Topics, tones, formats, assumptions, or behaviors that miss the mark with me.

Format:
Use bold section headers for each category. Within each category, list one entry per line, sorted oldest to newest. Format each line as:
[YYYY-MM-DD] - Entry content here.
Use [unknown] if no date is available.

Output Rules:
- Wrap the entire export in a single code block for easy copying into another AI.
- Be comprehensive. Include specific names, project titles, frameworks, and real examples — not generalizations.
- The goal is that any AI reading this output should feel like it already knows me and can work with me immediately.
- After the code block, note whether this is the complete export or if additional context remains.

HOW TO USE IT:

Step 1. Paste that prompt into your current AI tool and run it.
Step 2. Copy the entire output it generates.
Step 3. Open Claude at claude.ai and start a new conversation.
Step 4. Paste your exported profile and say: "This is my context. Read it and confirm you understand how to work with me."
Step 5. That's it. You're in. Claude already knows you.
